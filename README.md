# MonitorBlocker

MonitorBlocker is a lightweight Windows system tray application that allows users to block input on specific monitors by displaying a transparent overlay. It provides a simple way to toggle monitor blocking, lock or unlock all monitors, and unlock individual monitors with a mouse-based gesture. Built with Python, it is designed to be minimal, efficient, and easy to distribute as a standalone executable.

## Features

- **System Tray Interface**: Control monitor blocking from a system tray icon with a simple menu.
- **Toggle Monitors**: Lock or unlock individual monitors (e.g., "Toggle DISPLAY1 (1920x1080)").
- **Lock/Unlock All**: Quickly block or unblock all monitors.
- **Hold-to-Unlock**: Unlock a monitor by holding both left and right mouse buttons for 3 seconds (with blue background feedback).
